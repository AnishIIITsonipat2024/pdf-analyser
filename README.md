# 📄 AI PDF Analyzer

An AI-powered web application that allows users to upload PDF documents, extract text, generate intelligent summaries, and ask questions about the document using Natural Language Processing (NLP).

## 🚀 Live Demo

🔗 https://pdf-analyser-8y7k.onrender.com

---

## ✨ Features

- 📂 Upload PDF documents
- 📖 Extract text from PDFs
- 🤖 AI-powered document summarization
- ❓ Ask questions about the uploaded PDF
- ⚡ Fast and interactive web interface
- 📥 Download generated results

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/AnishIIITsonipat2024/AI-Meeting-Summarizer-.git
cd AI-Meeting-Summarizer-
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate

Windows

```bash
.venv\Scripts\activate
```

Linux/Mac

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run

```bash
python app.py
```

## 🖥️ Usage

1. Open the application.
2. Upload a PDF.
3. Wait for processing.
4. Read the extracted text.
5. Generate the summary.
6. Ask questions related to the document.
